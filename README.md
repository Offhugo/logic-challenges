# Logic & Algorithm Challenges

Este repositório é o meu laboratório pessoal de estudos focado em **Lógica de Programação**, **Estrutura de Dados** e **Resolução de Problemas**. 

Aqui documento o meu progresso em plataformas de desafios técnicos (como HackerRank), evoluindo desde os fundamentos da linguagem Java até algoritmos de complexidade intermediária.

## Por que este repositório é importante?
Desenvolver software vai além de frameworks; exige uma base sólida em lógica. Através desses desafios, busco:
- Refinar o uso de **Estruturas de Controle** (loops, condicionais).
- Dominar **Collections** e manipulação de arrays/listas.
- Praticar a eficiência de código e legibilidade.
